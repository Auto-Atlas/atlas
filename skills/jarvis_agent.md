# jarvis_agent

Delegates a task to the full OpenJarvis agent, which has real tools for web search,
files, shell and Python code, git, HTTP, databases, memory, messaging channels,
PDFs, and any connected accounts. Use it for anything beyond quick conversation —
research, lookups, file work, running code. It can take a while; say something like
"on it" first when natural. Delegate anything involving current information, facts
you are not certain of, or multi-step work. If the agent server is unreachable, tell
the user it isn't running.
