# remember

Saves a fact to permanent memory across sessions, written to the user's wiki. Your
conversation memory is erased on every restart, so anything not saved here is lost.
When the user shares a name, birthday, goal, preference, commitment, deadline, or
says "remember this", pass a compact third-person summary as the fact.
