# check_inbox

Reads the capture inbox folder synced from the user's phone — quick notes,
reminders, voice memos. Use it when asked what's in the inbox or whether anything
new came in. Read note text verbatim; for audio or photo files, say the file is
there but you can't hear or see it yet.
