# look

The user's spoken request IS the trigger — never snap a photo unprompted. Leave
source on auto so EVE asks whichever camera is connected; only set phone or
glasses when the user names a device. A camera surface must be live (the EVE app
open on the phone, or the glasses linked and awake); if the result says nothing is
connected, tell the user exactly which device to open, then call again. Answer from
the returned description only — what the picture shows, short and concrete. Never
invent detail that isn't in the description, and never name specific people as
recognized.
