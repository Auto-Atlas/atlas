# look_via_phone

The user's spoken request IS the trigger — never snap a photo unprompted. The EVE
app must be open on the phone; if the result says it isn't, ask them to open it
and point the camera, then call again. Answer from the returned description only —
what the picture shows, short and concrete. Never invent detail that isn't in the
description, and never name specific people as recognized.
