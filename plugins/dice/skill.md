# roll_dice

Rolls real dice. Use when the user asks for a dice roll, a random number from a
die, or wants chance to settle something. Arguments: `sides` (faces per die,
default 6), `count` (how many dice, default 1, max 20). Speak the individual
rolls and the total in one short sentence — "two dice: a 4 and a 6, total 10."
